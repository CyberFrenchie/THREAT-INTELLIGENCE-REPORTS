Each report investigates malicious infrastructure, malware families, and associated indicators of compromise (IOCs) using publicly available tools and data sources.

## Methodology

Each report follows a structured intelligence process:

1. **Trigger** — IOC identified via threat feeds (ThreatFox, VirusTotal, etc.)
2. **Infrastructure Analysis** — WHOIS, Shodan, geolocation correlation
3. **Malware Analysis** — Behavioural indicators, hash analysis, sandbox output
4. **IOC Extraction** — IP addresses, domains, file hashes
5. **Recommendations** — Defensive actions for detection and containment

---

## Tools Used

- [Shodan](https://www.shodan.io) — Internet-wide port and service scanning
- [VirusTotal](https://www.virustotal.com) — Malware hash and URL analysis
- [ThreatFox](https://threatfox.abuse.ch) — IOC feed and malware intelligence
- [IPInfoDB](https://www.ipinfodb.com) — Geolocation correlation
- WHOIS — Infrastructure ownership and abuse contact lookup
